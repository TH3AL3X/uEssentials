**How to Install:**

- Download the latest [release](https://github.com/TH3AL3X/uEssentials/releases).
- Extract contents to the server's Rocket folder. _(The Libraries and Plugins folders, to "../Servers/ServerName/Rocket/Plugins".)_
- Have fun!
